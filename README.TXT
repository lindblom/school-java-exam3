PROJECT TITLE: Vocabulary Trainer
PURPOSE OF PROJECT: Ett gloshantering och glostest program.
VERSION or DATE: 2012-11-09
HOW TO START THIS PROJECT: Kompilera projektets javafiler sedan kör java VocabularyTrainer i projektmappen i terminalen.
AUTHORS: Christopher Lindblom
USER INSTRUCTIONS:
När du startat VocabularyTrainer via kommandot java VocabularyTrainer i terminalen kommer du till huvudmenyn.
I huvud menyn har du tre alternativ, de är:
    test - för att köra ett glostest
    hantera - för att hantera glosboken
    avsluta - för att avsluta programmet

TEST:
Väljer du test kommer 10 av glosorna i glosboken att slumpas fram och du kommer att få ett test på skärmen.
Testet går till så att du frågas om ett ord åt gången till du svarat på samtliga.
Du har tre chanser på dig att komma fram till rätt svar.
Där på får du fram vilka du svarat rätt/fel på och rätt svar skrivs ut.

HANTERA:
I hanteramenyn har du fyra alternativ:
    lista - skriver ut orden som finns i glosboken
    lägg till - startar en dialog för att lägga till ord
    ta bort - startar en dialog för att ta bort ord (0 för att ångra)
    tillbaka - avslutar hanteringen och tar dig tillbaka till huvudmenyn
